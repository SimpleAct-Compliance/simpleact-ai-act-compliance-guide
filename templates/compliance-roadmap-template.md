# Compliance Roadmap Template

Use this roadmap to turn EU AI Act compliance into an operational sequence.

## Stage 1: Inventory

- identify systems
- assign owners
- record dependencies

## Stage 2: Classification

- screen prohibited practices
- assess high-risk relevance
- document assumptions

## Stage 3: Governance

- define approvals
- assign oversight
- define escalation

## Stage 4: Documentation

- create evidence records
- link templates
- capture decision logs

## Stage 5: Monitoring

- define review cadence
- define trigger events
- define outputs and reporting
